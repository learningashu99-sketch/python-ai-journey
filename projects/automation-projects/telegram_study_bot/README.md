# 🚀 FocusFlow Bot

A Telegram productivity bot designed for students to manage tasks, stay focused using Pomodoro sessions, maintain study streaks, and track productivity.

---

# 🎯 Project Overview

FocusFlow helps students:

* Organize study tasks
* Set priorities
* Track deadlines
* Receive automatic reminders
* Maintain study streaks
* Use Pomodoro focus sessions
* Monitor productivity statistics

---

# ✨ Features Implemented

## 📋 Task Management

* Add tasks
* View tasks
* Complete tasks
* Multi-user support

---

## 🔥 Priority System

* High Priority
* Medium Priority
* Low Priority

Commands:

```text
/setpriority
/highpriority
```

---

## ⏰ Due Date & Reminder System

* Set due dates
* Upcoming tasks
* Overdue tasks
* Automatic reminders

Commands:

```text
/setdue
/upcoming
/overdue
```

---

## 🔥 Study Streak System

Tracks daily task completion consistency.

Command:

```text
/streak
```

---

## 🍅 Pomodoro Focus Timer

* Custom Pomodoro sessions
* One active session per user
* Automatic completion notification

Commands:

```text
/pomodoro
/pomodorostats
```

---

## 📊 Productivity Analytics

Provides a productivity dashboard including:

* Current streak
* Completed Pomodoros
* Active tasks
* High-priority tasks
* Due date statistics

Command:

```text
/analytics
```
