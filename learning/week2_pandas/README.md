# 🎓 Student Dataset Analyzer

## 📌 Problem Statement

Analyze student performance data to extract meaningful insights such as top performers, average scores, and rankings.

---

## 📊 Dataset

The dataset contains:

* Student names
* Marks in multiple subjects

---

## ⚙️ Features

* Load and inspect dataset
* Find top students in each subject
* Calculate average marks
* Rank students based on performance
* Identify best overall student
* Filter high-performing students
* Assign performance labels (Excellent, Good, Needs Improvement)

---

## 🧠 Concepts Used

* Pandas DataFrame operations
* Filtering and selection
* Aggregation functions
* Data cleaning techniques

---

## ▶️ How to Run

```bash
python student_analyzer.py
```

---

## 📈 Output

* Ranked list of students
* Top performers in each subject
* Summary statistics of dataset

---

## 💡 Key Learnings

* Handling real-world datasets
* Applying data analysis techniques
* Writing structured and readable code
